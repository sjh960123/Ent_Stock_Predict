# 엔터테이먼트 주가 예측 프로젝트

## 데이터 출처
  주가 데이터: investing.com
  경제 지표: 국가 통계 포털(KOSIS)
  재무제표: opendart(전자공시 시스템)
  각 기업의 이슈: 네이버 뉴스 크롤링
  
## 데이터 시각화 및 분석
  각 기업의 주가와 경제지표, 재무제표, 기업의 이슈간 관계에 대해 시각화
  각 기업으 주가의 등락과 관련이 있는 확인
  
## 머신 러닝 기반 주가 예측
  scikit-learn 기반의 skforecast 라이브러리 사용
  여러 Regressor 모델을 사용하여 각 기업의 주가 예측 진행
